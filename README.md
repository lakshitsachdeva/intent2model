# Intent2Model

LLM-Guided AutoML Agent - Upload a CSV, chat with the AI, get a trained model.

## Quick Start

### Backend
```bash
cd backend
pip install -r ../requirements.txt
export GEMINI_API_KEY=your_key_here
python main.py
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```

Visit http://localhost:3000
