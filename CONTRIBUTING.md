# Contributing to drift

drift is open source. Contributions are welcome.

## How to contribute

1. **Fork** the repo
2. **Create a branch** (`git checkout -b feature/your-idea`)
3. **Make changes** and add tests if relevant
4. **Commit** with clear messages
5. **Push** and open a Pull Request

## Development setup

```bash
git clone https://github.com/lakshitsachdeva/intent2model.git
cd intent2model
pip install -r requirements.txt
pip install -e .  # editable install of drift-ml
```

For the web UI:
```bash
cd frontend && npm install && npm run dev
```

## Code style

- Python: follow PEP 8
- Be concise
- Add docstrings for public APIs

## Questions?

Open an issue. We'll respond.
